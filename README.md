# THE ATELIER — Design Studio Café

A luxury artisan café website inspired by architecture studios, design ateliers and creative workshops. Coffee, craftsmanship and spatial design merge into a gallery-quality digital experience.

## Brand

- **Name:** THE ATELIER
- **Concept:** Design Studio, Architecture, Creative Process, Minimalism, Gallery
- **Materials:** Concrete, Wood, Craftsmanship

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Paper White | `#F6F5F2` | Backgrounds |
| Concrete | `#C7C4BD` | Borders, muted text |
| Graphite | `#303030` | Body text |
| Oak | `#9C7545` | Accents, warmth |
| Ink | `#1C1C1C` | Headings, dark bg |
| Steel | `#9DA2A5` | Labels, tertiary |
| Accent Blue | `#4D83C8` | CTAs, highlights |

## Typography

- **Display:** Neue Haas Grotesk Display
- **Body:** Suisse International
- **Mono:** IBM Plex Mono

## Tech Stack

- Vite + React + TypeScript
- SCSS (Sass)
- GSAP (animations)
- Lenis (smooth scroll)
- SplitType (text splitting)
- Three.js (3D basics)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── components/       # React components
│   ├── Loader/       # Loading animation
│   ├── Navigation/   # Floating nav
│   ├── Hero/         # Hero section
│   ├── Philosophy/   # Editorial manifesto
│   ├── CreativeProcess/  # Design timeline
│   ├── SignatureCoffee/  # Coffee menu cards
│   ├── MaterialGallery/  # Masonry gallery
│   ├── TheWorkshop/  # 3D workbench
│   ├── DesignerJournal/  # Editorial articles
│   ├── StudioTeam/   # Team portraits
│   ├── Visit/        # Location info
│   ├── Reservation/  # Booking form
│   ├── Footer/       # Footer with credits
│   └── Cursor/       # Custom cursor
├── hooks/            # Custom React hooks
├── animations/       # GSAP animations
├── styles/           # SCSS variables and globals
└── utils/            # Constants and helpers
```
